## 1. 产品概述

地质勘探矢量数据 3D 建模交互系统是一个面向地质勘探行业的专业三维可视化平台，基于 Three.js + GeoServer + PostgreSQL 技术栈构建。系统加载地质勘探点位与地层矢量数据，在前端生成逼真的 3D 地层模型，支持视角漫游、距离/厚度量测、点位标注等交互操作，后端提供数据查询与坐标换算接口，空间数据库存储勘探数据，实现前后端、3D 引擎与数据库的完整联调。

- 目标用户：地质勘探工程师、地质数据分析师、科研人员
- 核心价值：将传统二维地质数据转化为直观的三维可视化模型，提升地质分析与决策效率

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 地质工程师 | 账号密码注册 | 浏览3D模型、量测标注、数据查询 |
| 数据管理员 | 管理员授权 | 数据导入导出、坐标转换、数据管理 |

### 2.2 功能模块

1. **主页面**：3D地形场景、工具栏、图层控制面板、数据面板
2. **数据管理页**：勘探数据列表、数据导入导出、坐标批量转换

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 主页面 | 3D地形建模 | 加载DEM高程数据生成3D地形网格，支持多分辨率地形渲染，纹理贴图叠加 |
| 主页面 | 矢量数据解析 | 解析GeoJSON/WFS矢量数据，在3D场景中渲染勘探点位、地层边界线、地质剖面 |
| 主页面 | 地质坐标转换 | 支持WGS84/GCJ02/BD09/西安80/北京54等坐标系互转，实时坐标显示与换算 |
| 主页面 | 视角漫游控制 | 轨道控制器旋转/缩放/平移，预设视角切换，飞行漫游路径 |
| 主页面 | 距离量测 | 在3D场景中点击两点测量空间距离、水平距离、垂直距离 |
| 主页面 | 厚度量测 | 选择地层面测量地层厚度，支持法线方向厚度计算 |
| 主页面 | 点位标注 | 在3D场景中添加文字标注、图钉标记，支持标注属性编辑与导出 |
| 主页面 | 图层控制 | 控制地形/点位/地层/标注等图层显示隐藏，透明度调节 |
| 主页面 | 数据面板 | 显示选中要素属性信息，勘探孔详细数据查看 |
| 数据管理页 | 勘探数据列表 | 分页展示勘探点位数据，支持搜索过滤与排序 |
| 数据管理页 | 数据导入导出 | CSV/GeoJSON格式数据批量导入导出 |
| 数据管理页 | 坐标批量转换 | 批量坐标转换任务，转换结果下载 |

## 3. 核心流程

用户进入系统后，通过主页面3D场景浏览地质数据。核心流程为：加载数据 → 生成3D模型 → 交互操作 → 量测标注 → 数据导出。

```mermaid
flowchart TD
    "用户登录" --> "进入主页面"
    "进入主页面" --> "加载地形与矢量数据"
    "加载地形与矢量数据" --> "生成3D地层模型"
    "生成3D地层模型" --> "视角漫游浏览"
    "生成3D地层模型" --> "距离/厚度量测"
    "生成3D地层模型" --> "点位标注"
    "距离/厚度量测" --> "量测结果展示"
    "点位标注" --> "标注属性编辑"
    "标注属性编辑" --> "导出标注数据"
    "视角漫游浏览" --> "选中要素查看属性"
    "选中要素查看属性" --> "坐标换算"
```

数据管理页流程：用户进入数据管理页 → 查看勘探数据列表 → 导入新数据/导出数据 → 批量坐标转换。

```mermaid
flowchart TD
    "进入数据管理页" --> "查看勘探数据列表"
    "查看勘探数据列表" --> "搜索过滤数据"
    "查看勘探数据列表" --> "导入CSV/GeoJSON数据"
    "查看勘探数据列表" --> "导出选中数据"
    "查看勘探数据列表" --> "批量坐标转换"
```

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：深蓝灰 (#1a2332) 配地质暖橙 (#e87c3e) 强调色，营造专业地质勘探氛围
- 辅助色：岩层灰 (#4a5568)、矿石绿 (#38a169)、水晶蓝 (#4299e1)
- 按钮风格：圆角矩形，hover时微弱发光效果，主操作按钮带地质暖橙色底
- 字体：标题使用 Rajdhani（科技感），正文使用 Source Sans 3
- 布局风格：左侧图层面板 + 中央3D视口 + 右侧属性面板，顶部工具栏
- 图标风格：线性图标，基于 Lucide React

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 主页面 | 3D视口 | 全屏Three.js Canvas，深色背景，HDR环境光照，地形网格+纹理 |
| 主页面 | 工具栏 | 顶部水平工具栏，图标按钮分组：漫游/量测/标注/视角，活跃工具高亮 |
| 主页面 | 图层面板 | 左侧可折叠面板，树形图层列表，复选框+透明度滑块 |
| 主页面 | 属性面板 | 右侧可折叠面板，表格展示选中要素属性，坐标信息实时显示 |
| 主页面 | 量测结果 | 浮动卡片，显示距离/厚度数值，支持复制 |
| 主页面 | 标注弹窗 | 模态框，标注名称/描述/类型输入表单 |
| 主页面 | 坐标转换面板 | 底部抽屉，源坐标系/目标坐标系下拉选择，坐标值输入输出 |
| 数据管理页 | 数据表格 | 全宽表格，列排序/筛选，分页控件，行选择复选框 |
| 数据管理页 | 导入导出区 | 工具栏按钮组，拖拽上传区域，进度条 |
| 数据管理页 | 批量转换区 | 表单+表格组合，坐标系选择，转换状态列 |

### 4.3 响应式设计

- 桌面优先设计，3D视口最小宽度 1024px
- 侧边面板可折叠，小屏自动收起
- 工具栏响应式分组，窄屏使用下拉菜单
- 触摸屏支持手势操作（双指缩放、单指旋转）

### 4.4 3D场景指引

- 环境：暗色空间背景，模拟地下勘探氛围，微弱环境光
- 灯光：方向光模拟顶部照明，环境光提供基础亮度，点光源标注重要区域
- 相机：透视相机，45°FOV，近裁剪面0.1，远裁剪面10000，轨道控制器
- 构图：地形居中，默认俯视45°视角，地层面环绕地形展示
- 交互：鼠标左键旋转、右键平移、滚轮缩放，双击聚焦要素
- 动画：场景加载时地形渐现动画，量测线绘制动画，标注弹出动画
- 后处理：边缘发光（Bloom）用于高亮选中要素，环境光遮蔽（SSAO）增强深度感
