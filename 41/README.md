# 光伏阵列组串状态监测与远程运维管控系统

基于微服务架构的光伏电站监测系统，实现设备拓扑可视化、实时数据采集、异常分析和远程运维。

## 系统架构

```
┌─────────────────────────────────────────────────────────────┐
│                     前端拓扑监控面板                          │
│              (React + TypeScript + ECharts)                 │
└────────────────────┬────────────────────────────────────────┘
                     │
┌────────────────────▼────────────────────────────────────────┐
│                     API 网关 / Nginx                         │
└─────────┬───────────────────┬───────────────────┬────────────┘
          │                   │                   │
┌─────────▼────────┐ ┌────────▼───────┐ ┌────────▼──────────┐
│  数据采集网关    │ │ 组串数据分析   │ │ 远程指令下发模块   │
│  (FastAPI+MQTT)  │ │  (FastAPI)     │ │  (FastAPI+MQTT)   │
└─────────┬────────┘ └────────────────┘ └────────┬──────────┘
          │                                       │
┌─────────▼────────┐                    ┌────────▼──────────┐
│  异常告警推送    │                    │   消息队列(MQTT)   │
│  (WebSocket)     │                    └────────┬──────────┘
└──────────────────┘                             │
                                                 │
                                    ┌────────────▼───────────┐
                                    │  边缘节点 / 嵌入式设备  │
                                    │  (光伏组串采集器)      │
                                    └────────────────────────┘
```

## 模块说明

| 模块 | 端口 | 技术栈 | 说明 |
|------|------|--------|------|
| frontend | 3000 | React + TypeScript | 前端监控面板 |
| gateway | 8000 | FastAPI + MQTT | 数据采集网关 |
| analysis | 8001 | FastAPI + Pandas | 数据分析服务 |
| command | 8002 | FastAPI + MQTT | 指令下发服务 |
| alert | 8003 | FastAPI + WebSocket | 告警推送服务 |

## 快速开始

```bash
# 启动所有服务
docker-compose up -d

# 或单独启动
cd frontend && npm install && npm start
cd ../gateway && pip install -r requirements.txt && uvicorn main:app --reload --port 8000
```
