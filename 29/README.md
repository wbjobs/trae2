# 虚拟沙盘 - 地形侵蚀演化模拟联机游戏

一个基于 Three.js 和 Node.js 的多人联机地形侵蚀演化模拟游戏，支持水流侵蚀、风力侵蚀物理模拟，可自定义参数配置，支持本地+云端双存档。

## 功能特性

### 🌍 物理模拟
- **水流侵蚀** - 模拟降雨、水流、泥沙搬运和沉积过程
- **风力侵蚀** - 模拟风蚀、磨蚀、跳跃搬运和悬浮搬运
- **实时演化** - 地形随时间动态变化

### 👥 多人联机
- **实时同步** - 所有玩家共享同一地形状态
- **玩家列表** - 显示在线玩家
- **聊天系统** - 玩家间实时通讯

### ⚙️ 自定义配置
- **参数调节** - 可调节降雨速率、侵蚀强度、风力大小等数十种参数
- **预设场景** - 沙漠、三角洲、山地、海岸等预设地形
- **地形生成** - 自定义尺寸、高度、随机种子

### 💾 存档系统
- **本地存档** - 保存到本地文件系统
- **云端存档** - 服务器内存存档（可扩展为数据库）
- **同步功能** - 本地存档同步到云端

## 快速开始

### 环境要求
- Node.js >= 16.0.0
- npm 或 yarn

### 安装依赖

```bash
# 安装服务端依赖
cd server
npm install

# 安装客户端依赖
cd ../client
npm install
```

### 启动服务

#### 方式一：分别启动

```bash
# 终端 1：启动服务端
cd server
npm start

# 终端 2：启动客户端
cd client
npm run dev
```

#### 方式二：使用启动脚本（Windows）

双击根目录下的 `start.bat` 即可一键启动服务端和客户端。

### 访问游戏

客户端启动后，浏览器会自动打开 `http://localhost:5173`

1. 输入服务器地址（默认 `http://localhost:3000`）
2. 输入玩家名称
3. 点击「连接服务器」

## 操作说明

### 视角控制
- **鼠标左键拖动** - 旋转视角
- **鼠标滚轮** - 缩放
- **鼠标右键拖动** - 平移视角（可选）

### 控制面板
- **模拟控制** - 开关模拟、调节时间倍速
- **水流侵蚀** - 启用水流、调节降雨和侵蚀参数
- **风力侵蚀** - 启用风蚀、调节风力和风向
- **预设场景** - 快速应用预设地形配置
- **地形生成** - 重新生成随机地形
- **存档管理** - 保存、加载、删除存档

## 配置说明

### 水流侵蚀参数
| 参数 | 范围 | 说明 |
|------|------|------|
| 降雨速率 | 0-2 | 降雨量大小，影响水流量 |
| 侵蚀强度 | 0-1 | 水流侵蚀地形的强度 |
| 沉积速率 | 0-1 | 泥沙沉积的速度 |
| 侵蚀半径 | 1-8 | 每个水滴影响范围 |

### 风力侵蚀参数
| 参数 | 范围 | 说明 |
|------|------|------|
| 风力强度 | 0-3 | 风速大小 |
| 风向角度 | 0-360° | 风吹来的方向 |
| 磨蚀速率 | 0-0.5 | 风对地表的磨蚀强度 |

### 地形生成参数
| 参数 | 范围 | 说明 |
|------|------|------|
| 地形尺寸 | 64-256 | 地形网格大小 |
| 高度倍率 | 10-150 | 地形相对高度 |
| 随机种子 | 任意 | 地形生成随机种子 |

## 技术栈

### 服务端
- **Node.js** - 运行环境
- **Express** - Web 框架
- **Socket.IO** - 实时通信
- **Noise.js** - Perlin 噪声生成

### 客户端
- **Three.js** - 3D 渲染
- **Socket.IO Client** - 网络通信
- **Vite** - 构建工具

## 平台适配

### PC 端
- 支持 Chrome、Firefox、Edge 等现代浏览器
- 推荐使用鼠标操作

### 云游戏平台
- 可通过云游戏流化技术部署
- 支持低延迟操作
- 无需本地安装

## API 接口

### HTTP 接口
- `GET /health` - 服务健康检查
- `GET /api/saves` - 获取存档列表

### Socket.IO 事件
- `join` - 加入游戏
- `playerMove` - 玩家移动
- `chatMessage` - 发送消息
- `updateConfig` - 更新配置
- `regenerateTerrain` - 重新生成地形
- `saveGame` / `loadGame` - 存档操作

## 性能优化建议

1. **地形尺寸** - 较低的尺寸（64/128）可获得更好的性能
2. **粒子数量** - 减少风力粒子数量可提高帧率
3. **同步频率** - 降低同步率可减少网络带宽
4. **浏览器** - 推荐使用 Chrome 浏览器获得最佳体验
