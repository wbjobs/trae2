# 深海探测设备仿生动作模拟仿真游戏

## 项目简介

这是一个基于 Three.js 的深海探测设备仿生动作模拟仿真游戏，支持单机离线和局域网联机两种模式。游戏模拟了深海环境中探测设备的仿生游动、避障、数据采集等动作。

## 项目结构

```
deep-sea-exploration-simulator/
├── client/                    # 客户端代码
│   ├── index.html             # 游戏入口页面
│   ├── style.css              # 样式文件
│   ├── renderer.js            # 3D渲染引擎
│   ├── gameClient.js          # 游戏客户端逻辑
│   └── main.js                # 主程序入口
├── server/                    # 服务端代码
│   ├── server.js              # 服务器主程序（含HTTP和WebSocket）
│   └── physicsEngine.js       # 物理引擎和动作计算
├── shared/                    # 共享模块
│   ├── config.js              # 全局配置
│   ├── utils.js               # 工具函数
│   ├── sensors.js             # 传感器数据模拟
│   ├── collision.js           # 碰撞检测
│   └── mission.js             # 任务剧情系统
├── package.json               # 项目配置
└── README.md                  # 项目说明
```

## 核心模块说明

### 1. 客户端模型渲染 (client/renderer.js)
- 使用 Three.js 进行3D场景渲染
- 实现仿生探测设备模型（身体、尾鳍、胸鳍、螺旋桨）
- 实现深海环境（水面、海底、障碍物、样本）
- 支持跟随视角和自由视角切换
- 实时声呐可视化

### 2. 动作逻辑服务端 (server/physicsEngine.js)
- 统一运算物理逻辑
- 仿生动作模拟（尾鳍摆动、胸鳍转向、身体俯仰）
- 浮力、重力、水阻力等物理效果
- 能量消耗系统

### 3. 传感器数据模拟模块 (shared/sensors.js)
- 声呐传感器（多波束障碍物检测）
- 压力传感器（深度计算）
- 温度传感器
- 氧气传感器
- 速度、朝向、姿态传感器

### 4. 碰撞检测模块 (shared/collision.js)
- 边界碰撞检测
- 障碍物碰撞检测
- 载具间碰撞检测
- 地形碰撞检测
- 碰撞伤害计算

### 5. 任务剧情模块 (shared/mission.js)
- 5个主线任务（浅海探索→珊瑚礁测绘→深海下潜→沉船搜索→数据回收）
- 任务进度追踪
- 样本收集系统
- 剧情对话系统

## 运行方法

### 前置要求
- Node.js 14+
- npm 或 yarn

### 安装依赖
```bash
npm install
```

### 启动服务端（联机模式需要）
```bash
npm start
```
服务端将在 `http://localhost:3000` 启动

### 开始游戏

#### 单机模式
1. 直接打开 `client/index.html`
2. 点击"单机离线模式"
3. 开始游戏

#### 联机模式
1. 启动服务端（见上文）
2. 打开浏览器访问 `http://localhost:3000`
3. 输入玩家名称，选择颜色
4. 点击"加入游戏"
5. 开始游戏

### 局域网联机
1. 确保服务端和客户端在同一局域网
2. 在客户端输入服务端的局域网IP地址，例如 `ws://192.168.1.100:3000`
3. 点击"加入游戏"

## 技术特性

- **前后端分离架构**：服务端统一运算物理逻辑，多客户端同步设备状态
- **WebSocket实时通信**：支持最多8名玩家同时在线
- **3D渲染**：基于Three.js的高质量深海环境渲染
- **物理模拟**：真实的水下物理效果（浮力、阻力、重力）
- **仿生动作**：模拟鱼类游动的尾鳍摆动、胸鳍转向等动作
- **传感器模拟**：多种传感器数据实时计算和显示
- **任务系统**：丰富的任务剧情和目标系统

## 游戏玩法

1. **探索**：使用WASD控制设备在深海中探索
2. **避障**：通过声呐检测障碍物，避免碰撞
3. **采集**：接近发光的样本自动收集
4. **任务**：按M打开任务面板，接受并完成任务
5. **升级**：完成任务获得积分奖励

## 配置修改

可以通过 `shared/config.js` 修改游戏参数：
- 世界大小、深度
- 载具速度、加速度
- 传感器范围、精度
- 网络端口、同步频率
- 渲染效果等
